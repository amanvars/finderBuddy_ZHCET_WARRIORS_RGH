Instructions

1. To place an Order, please type: #place_order /product_name/quantity/color(if any)

2. To change an Order, please type: #change_order /order_id/quantity(to change)

3. To review an Order, please type: #review_order /order_id

4. To cancel an Order, please type: #cancel_order /order_id

5. To know your previous Order, please type: #previous_order

6. To check what all items are about to finish, please type: #check

7. For help, please type: #help

8. Other than this,you can also just type in the keywords of your query

9. like want/need/find/reorder along with the order id and product name

10. pattern is sensitive and use of small case letters is recommended
